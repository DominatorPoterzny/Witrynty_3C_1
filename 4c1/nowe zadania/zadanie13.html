Magiczny świat HTML, CSS i JavaScript

HTML: Fundamenty Magicznego Domu

HTML, najstarszy z trójki, była mądra i pełna energii. Jej zadaniem było budowanie szkieletów stron internetowych. Niczym architekt, projektowała i konstruowała budynki - definiowała gdzie mają być umieszczone nagłówki, akapity, obrazy i przyciski. Dzięki niej wszystkie strony miały solidne podstawy

CSS: Artysta i Dekorator

CSS, młodsza i pełna kreatywnoiści, była artystką, która kochała kolory, czcionki i układy. Za pomocą magicznych pędzli, zwanych kaskadowymi arkuszami stylów, CSS nadawała każdemu elementowi estetyczny wygląd

Na przykład, oto jak CSS nadaje kolor tła:

body{
        background-color: lightblue;
        }

CSS również porządkowała zawartość strony, nadając styl nagłówkom i akapitom:

h1{
        color: darkblue;
        font-family: Arial, Helvetica, sans-serif;
        text-align: center;
        }
        p{
        color: darkgray;
        font-size: 16px;
        line-height: 1.5;
        margin: 20px;
        }

JavaScript: Czarodziejka Animacji

Na końcu pojawiła wię JavaScript, najmłodsza z przyjaciółek, ale również najdynamiczniejsza. Była czarodziejką, która dodawała interaktywności i animacje. Razem z CSS stworzyła niesamowiny efekt przycisków: 

.button {
        background-color: darkblue;
        color: white;
        padding: 10px 20px;
        border: none;
        cursor: pointer;
        transition: background-color 0.3s;
        }
        .button:hover {
        background-color: white;
        color: navy;
        box-shadow: 2px 2px 2px 2px rgba(0, 0, 0, 0.2);
        border-left: 1px solid lightgrey;
        border-top: 1px solid lightgrey;
        }
