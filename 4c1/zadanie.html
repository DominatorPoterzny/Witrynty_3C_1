KASKADOWE ARKUSZE STYLÓW - CSS


CSS: (Cascading Style Sheets) to język, który wykorzystuje się do opisania gdzie powinny znajdować się elementy strony na ekranie, jaki kolor czcionki powinien być ustawiony I czy teskt na stronie nie powinien być tylko po prawej stronie (jest to tylko pare przykładowych użyć CSS -a). Jest on używany razem z HTML​

Sposoby dodawania stylu:

  • W osobnym pliku CSS - jest najdalej w hierarchi dodawania styli; Należy dodać element "link", który połączy plik HTML z plikiem CSS.
  • W samym pliku HTML: należy wykorzystać element "style" w którym dodamy style dla strony; Jest 2 w hierarchi dodawania styli.
  • Przy samym elemencie INLINE: jest najwyżej w hierarchi dodawania styli
